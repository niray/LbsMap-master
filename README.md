
## 利用百度地图快速实现支付宝的“到位”功能。


* 基于LBS精确获取地理位置的周边数据
* 地图动态显示网络图标
* 支持多点聚合与分离
* 点击图标动态放大选中效果
* 移动、放大、缩小地图后重新更新数据
* 聚合图标点击动画展开
* 根据实际距离调整地图层级为屏幕大小
* 分页刷新
